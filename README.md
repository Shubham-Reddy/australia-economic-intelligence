# ![Australia](https://flagcdn.com/w40/au.png) Australia Economic Intelligence Platform
AI powered national economic dashboard providing real time insights across rental markets, job trends, cost of living, property analysis and economic advisory across Australia.

## 🔴 Live Demo
Coming soon

## 🚀 Features
- 🏠 Rental Crisis Tracker with ML predictions
- 💼 Job Market Intelligence
- 💰 Cost of Living Index
- 🏡 Property vs Rental Analysis with Buy vs Rent Calculator
- 🤖 AI Economic Advisor powered by Groq LLaMA 3.3 70B
- 🗺️ Interactive Australia maps on all modules
- 📊 Suburb level rental search
- 📈 Live RBA cash rate and ABS economic indicators

## 🛠️ Tech Stack
Python, Streamlit, Plotly, Folium, Scikit-learn, Groq LLaMA, BeautifulSoup, Pandas

## 📊 Data Sources
- RBA — Live cash rate scraping
- ABS — Live inflation and unemployment
- CoreLogic — Rental data June 2026
- ABS RPPI — Property prices March 2026

## 🔧 Run Locally
git clone https://github.com/Shubham-Reddy/australia-economic-intelligence
cd australia-economic-intelligence
pip install -r requirements.txt
streamlit run app.py
